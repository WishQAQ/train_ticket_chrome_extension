# train_ticket_chrome_extension
`vue-cli3` + `webpack4` + `element-ui`

## 环境安装
```
npm install
```

### 开发环境编译并热更新
```
npm run serve
or
npm run build-watch
```

### 生产环境打包
```
npm run build
```

### 分析包组件大小
```
npm run analyze
```

### Lints and fixes files
```
npm run lint
```
